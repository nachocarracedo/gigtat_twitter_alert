## Scrape festivals tweeter for alerts

## USE 

1) Get your twitter keys:

* Create a Twitter user account if you don't have one.
Go to https://apps.twitter.com/ and log in with your Twitter user account. These steps give you a Twitter dev account under the same name as your user account:
* Click “Create New App”
* Fill out the form, agree to the terms, and click “Create your Twitter application”
* In the next page, click on “Keys and Access Tokens” tab, and copy your “API key” and “API secret”. Scroll down and click “Create my access token”, and copy your “Access token” and “Access token secret”. 

2) Add your keys to settings.py

3) Add email info to settings.py

4) Make sure you are running Python 3.X and install requirements.txt (pip install -r /path/to/requirements.txt)

4) Run script
